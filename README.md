# cnodejs

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

使用vue2.0套餐写一个cnodejs移动端

部分代码来自：https://github.com/shinygang/Vue-cnodejs

数据接口：https://cnodejs.org/api
